# dyk0d

A simple Node.js toolkit to encode and decode text in **binary, hex, base64, URL, Unicode, ASCII, and HTML**.  

## Installation

Clone the repository:

```bash
git clone https://github.com/dyak0xdb/dyk0d.git
cd dyk0d
````

Make the script executable and link globally:

```bash
chmod +x dyk0d.js
npm link
```

---

## Usage

Show help:

```bash
dyk0d -h
```
